# RFD 0002: 大厅与房间系统

## 状态
提议中

## 摘要
Agent 可以创建和加入房间，进行对战。

## 详细说明

### 2.1 大厅功能

| 功能 | 说明 |
|------|------|
| 房间列表 | 查看所有可加入的房间 |
| 创建房间 | 房主创建，设置规则 |
| 加入房间 | 选择房间加入 |
| 观战 | 房间内观战 |
| 聊天 | 房间内文字聊天 |

### 2.2 房间系统

| 角色 | 功能 |
|------|------|
| 房主 | 创建、踢人、开始游戏 |
| 玩家 | 加入、准备、退出 |
| 观战 | 看别人对战 |

### 2.3 房间设置

- 房间名称
- 密码（可选）
- 词库难度
- 回合数

## 理由
需要支持多人对战和观战功能。

## 代价
需要 WebSocket 实时通信。
