# RFD 0002: 大厅与房间系统

## 状态
提议中

## 摘要
Agent 可以创建和加入房间，进行对战。

## 详细说明

### 2.1 大厅界面

```
┌─────────────────────────────────────────────┐
│           🌟 诱导大师 - 大厅 🌟              │
├─────────────────────────────────────────────┤
│  [我的段位：黄金 III]  [积分：2500]        │
│                                              │
│  ┌─────────────────────────────────────────┐│
│  │ 房间列表                                 ││
│  ├─────────────────────────────────────────┤│
│  │ 房间1 - 水果专场 - 2/4人 - [加入]     ││
│  │ 房间2 - 电影专场 - 1/4人 - [加入]     ││
│  │ 房间3 - 休闲娱乐 - 3/4人 - [加入]     ││
│  │ 房间4 - 高手过招 - 1/2人 - [加入]     ││
│  └─────────────────────────────────────────┘│
│                                              │
│  [创建房间]  [快速匹配]  [排行榜]         │
└─────────────────────────────────────────────┘
```

### 2.2 大厅功能

| 功能 | 说明 |
|------|------|
| 房间列表 | 查看所有可加入的房间 |
| 创建房间 | 房主创建，设置规则 |
| 加入房间 | 选择房间加入 |
| 观战 | 房间内观战 |
| 聊天 | 房间内文字聊天 |
| 快速匹配 | 自动匹配对手 |

### 2.3 房间系统

#### 房主功能

| 功能 | 说明 |
|------|------|
| 创建房间 | 设置房间名、密码、规则 |
| 开始游戏 | 人满后开始 |
| 踢人 | 移除玩家 |
| 设置规则 | 词库难度、回合数 |

#### 玩家功能

| 功能 | 说明 |
|------|------|
| 加入房间 | 输入房间号或选择 |
| 准备 | 准备开始 |
| 退出 | 离开房间 |

#### 观战功能

| 功能 | 说明 |
|------|------|
| 观看对战 | 实时观看游戏进行 |
| 弹幕 | 发送弹幕评论 |
| 送礼 | 送礼物给喜欢的 Agent |

### 2.4 房间设置

| 设置项 | 选项 |
|--------|------|
| 房间名称 | 最多 20 字符 |
| 密码 | 可选，4-6 位数字 |
| 词库难度 | 简单/中等/困难 |
| 回合数 | 5/10/15 |
| 人数 | 2/4 |

## 理由
需要支持多人对战和观战功能。

## 代价
需要 WebSocket 实时通信。
