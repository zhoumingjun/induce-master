# RFD 0003: 匹配与排名系统

## 状态
提议中

## 摘要
天梯排位系统，根据实力匹配对手。

## 详细说明

### 3.1 段位系统

| 段位 | 积分范围 | 图标 |
|------|----------|------|
| 青铜 | 0-999 | 🥉 |
| 白银 | 1000-1999 | 🥈 |
| 黄金 | 2000-2999 | 🥇 |
| 钻石 | 3000-3999 | 💎 |
| 王者 | 4000+ | 👑 |

### 3.2 匹配机制

| 规则 | 说明 |
|------|------|
| 段位匹配 | 相差不超过 1 个大段 |
| 积分匹配 | 积分相差不超过 200 |
| 等待超时 | 60 秒未匹配成功，放宽条件 |
| 快速匹配 | 可以选择跨段位匹配 |

### 3.3 积分计算

| 情况 | 得分 |
|------|------|
| 胜利（段位高于对手） | +15 |
| 胜利（段位相同） | +20 |
| 胜利（段位低于对手） | +25 |
| 失败 | -10 |
| 平局 | +0 |

### 3.4 排行榜

| 排行榜 | 说明 |
|--------|------|
| 全服排名 | 所有玩家积分排行 |
| 好友排名 | 好友积分排行 |
| 历史最高 | 最高段位排行 |

### 3.5 赛季制度

| 项目 | 说明 |
|------|------|
| 赛季时长 | 30 天 |
| 赛季奖励 | 段位专属头像框 |
| 赛季重置 | 保留段位，积分重置 |

## 理由
增加游戏粘性和竞技性。

## 代价
需要积分系统和数据库支持。
