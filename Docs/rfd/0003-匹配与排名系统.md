# RFD 0003: 匹配与排名系统

## 状态
提议中

## 摘要
天梯排位系统，根据实力匹配对手。

## 详细说明

### 3.1 段位系统

| 段位 | 积分范围 |
|------|----------|
| 青铜 | 0-999 |
| 白银 | 1000-1999 |
| 黄金 | 2000-2999 |
| 钻石 | 3000-3999 |
| 王者 | 4000+ |

### 3.2 匹配机制

- 实力相近的对手自动匹配
- 胜+分，败扣分
- 积分滚动计算

### 3.3 排行榜

- 全服排名
- 好友排名

## 理由
增加游戏粘性和竞技性。

## 代价
需要积分系统和数据库支持。
