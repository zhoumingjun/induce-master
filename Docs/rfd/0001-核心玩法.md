# RFD 0001: 关键词诱导战核心玩法

## 状态
提议中

## 摘要
每个 Agent 有一个关键词（自己不知道），需要通过对话诱导对手说出 ta 的关键词。

## 详细说明

### 1.1 基本规则

| 要素 | 设计 |
|------|------|
| 每个 Agent | 有一个关键词（自己不知道） |
| 每个 Agent | 知道对方的关键词 |
| 目标 | 诱导对方说出 ta 的关键词 |
| 胜利条件 | 诱导成功 + 对方先说出 |

### 1.2 词库分类

| 分类 | 难度 | 示例 |
|------|------|------|
| 水果 | 简单 | 苹果、香蕉、葡萄 |
| 动物 | 简单 | 猫、狗、兔子 |
| 城市 | 中等 | 北京、上海、成都 |
| 电影 | 中等 | 阿凡达、复联 |
| 成语 | 困难 | 画蛇添足、掩耳盗铃 |

### 1.3 判定机制

精确匹配、近似匹配、描述检测。

## 理由
这个玩法能体现 Agent 的"灵魂"（性格），同时人的作用是调教 Agent 的提示词。

## 代价
需要开发词库管理系统、判定引擎。
