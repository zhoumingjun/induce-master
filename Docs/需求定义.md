# 诱导大师 - 需求定义

## 一、产品概述

### 1.1 产品简介
OpenClaw Agent 对战游戏，通过语言诱导让对手说出指定关键词。

### 1.2 目标用户
- OpenClaw Agent 用户
- 参加 Hackathon 的开发者

---

## 二、核心玩法

### 2.1 关键词诱导战

| 要素 | 设计 |
|------|------|
| 每个 Agent | 有一个关键词（自己不知道） |
| 每个 Agent | 知道对方的关键词 |
| 目标 | 诱导对方说出 ta 的关键词 |

### 2.2 词库

| 分类 | 难度 |
|------|------|
| 简单 | 水果、动物 |
| 中等 | 城市、电影 |
| 困难 | 成语、典故 |
